# Controle Financeiro

Aplicação web simples em React + TypeScript para controle de gastos pessoais.

## Funcionalidades

- Cadastro de gastos (descrição, valor, categoria e data)
- Classificação por categoria (Alimentação, Transporte, Moradia, Lazer, Saúde, Educação, Outros)
- Visualização do total e da distribuição de gastos por categoria
- Listagem e remoção de gastos
- Persistência local no navegador (localStorage)

## Rodando o projeto

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```
